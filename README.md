## Files

The original dataset includes the following MATLAB `.mat` files:

- `QuadratureDistribution.mat`
- `RabiOscillations.mat`
- `ReflectionMeasurement.mat`
- `SpectrumAnalyzerMeasurement.mat`

## Dataset Description

According to the original dataset description, these files originate from:

- Frequency-domain measurements (Vector Network Analyzer and Spectrum Analyzer)
- Time-domain measurements (digitizer)

The measurements are performed on a microwave single-photon source and power sensor device based on a superconducting transmon qubit.

## Summary of Included Files

- **ReflectionMeasurement.mat**  
  Contains reflection measurements at the emission port of the qubit, including both off-resonance and on-resonance matrices.

- **SpectrumAnalyzerMeasurement.mat**  
  Contains spectrum analyzer traces measured for a VNA tone with corrected power values.

- **RabiOscillations.mat**  
  Contains mode-matched Rabi emission used to calibrate qubit drive pulses.

- **QuadratureDistribution.mat**  
  Contains mode-matched emission after preparing the qubit in ground, excited, and superposition states.

## Usage Note

These files can be read in both Python and MATLAB.  
The data structures can be interpreted as dictionaries of arrays or matrices with descriptive key names.
